# Traffic Vehicle Assets

Place **8 AI traffic vehicle PNG files** in this directory.

## Required Files:
- `sedan_gray.png` - Gray sedan
- `compact_white.png` - White compact car
- `suv_blue.png` - Blue SUV
- `truck_red.png` - Red truck
- `van_white.png` - White van
- `bus_yellow.png` - Yellow bus
- `sports_red.png` - Red sports car
- `motorcycle_black.png` - Black motorcycle (if available)

## Source:
Download from: https://kenney.nl/assets/pixel-vehicle-pack

## File Format:
- PNG images with transparency
- Top-down view
- Various sizes (sedans smaller, buses/trucks larger)
- Recommended size: 64x64 to 128x128 pixels

See `MANUAL_DOWNLOAD_INSTRUCTIONS.md` in the project root for detailed download steps.
