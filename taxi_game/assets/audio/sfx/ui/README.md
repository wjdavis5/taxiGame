# UI Sound Effects

Place **3 UI sound effect files** in this directory.

## Required Files:
- `button_click.wav` (or .mp3) - Short click sound for button presses
- `coin_collect.wav` (or .mp3) - Pleasant "cha-ching" sound for collecting coins
- `level_complete.wav` (or .mp3) - Upbeat, ascending tone for completing levels

## Source:
Generate using JSFXR: https://sfxr.me/

## File Format:
- WAV or MP3
- Short duration (0.1 - 0.5 seconds for clicks, 0.5 - 1.5 seconds for others)
- Mono or stereo

## Instructions:
See `MANUAL_DOWNLOAD_INSTRUCTIONS.md` in the project root for detailed generation steps.

## Tips:
- Button Click: Use "PICKUP/COIN" preset, find a short satisfying click
- Coin Collect: Use "PICKUP/COIN" preset, find a pleasant "cha-ching" sound
- Level Complete: Use "POWERUP" preset, find an upbeat ascending tone
