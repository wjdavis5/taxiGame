# Background Music

Place **1 background music file** in this directory for MVP.

## Required Files:
- `menu_music.mp3` - Upbeat background music for the main menu

## Source:
Download from Incompetech: https://incompetech.com/music/royalty-free/

## Recommended Tracks:
1. **Cipher** (Primary recommendation)
   - URL: https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1100525
   - Style: Electronic, upbeat (122 BPM)
   
2. **Pixel Peeker Polka** (Alternative)
   - URL: https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1100768
   - Style: Fun, retro
   
3. **Hyperfun** (Alternative)
   - URL: https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1100316
   - Style: Electronic dance (140 BPM)

## File Format:
- MP3
- Good quality (128kbps or higher)
